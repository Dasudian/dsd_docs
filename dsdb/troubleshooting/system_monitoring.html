System Monitoring

To assist with troubleshooting and performance analysis DSDB makes available statistical and diagnostic information about the system. For additional information see the README on GitHub.

  • See statistics for your installation with SHOW STATS
  • Show diagnostic information about your installation with SHOW DIAGNOSTICS
